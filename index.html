¿Cómo se integra JavaScript a HTML?.

Hay tres formas de integra JavaScript a html, son:

La primera forma es externa, es la mas recomendable. Esta en de enlace externo para cuando tangamos un archivo aparte ya cerrado y solo con las siguientes etiquetas se pudierán enlazar más fácil, en la etiqueta head se pone la etiqueta scrip con el atributo src y en ese atributo se posiciona el nombre del archivo scrip e indicando en la carpeta donde se guardo, y despues se cierra la etiqueta scrip y luego el head.

La segunda manera de enlazar un archivo Javascript en html también se hace en la etiqueta head, pero con la diferencia de que en esta parte debemos mostrar el código y no solo el archivo externo.

La tercera forma es mostrando el código de forma interna en la etiqueta body.
