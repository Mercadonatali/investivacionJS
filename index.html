¿Cómo es el manejo de las variables, tipos de datos y expresiones regulares en JS?.

El manejo de las variables se manejan guardando cualquier tipo de datos u información en javascript. Estas variables que asignamos le dan un nombre al valor, para reutilizarlo, actualizarlo o registrarlo. Al asignar las variables no se declaran el valor y por ello se utiliza el mismo nombre de variable para que podamos usar distintos tipos de datos. El manejo de tipo de datos se maneja realizando la ejecución de programa mientras se lleve a cabo la compilación. Hay 7 tipos de datos que son:
  • String: Este tipo de datos es una cadena de caracteres, a cada uno de los caracteres se le pone una posición, empezando por la posición 0.
  • Number: Este tipo de datos son numéricos, no tiene un tipo especifico para las comas flotantes y números enteros.
  • Boolean: Este tipo de datos representa una entidad lógica que puede tener los valore siguientes, true y false.
  • Null: Este tipo de datos tiene un valor null.
  • Udefined: Este tipo de dato se le asigna la variable que no tiene valor para que este le asigne el valor.
  • Symbol: Este tipo de datos tiene los valores únicos e inmutables. Estos valores se los puede utilizar como claves. Cada uno de estos valores tiene asignado un valor de tipo String o Undefined.
  • Object: Este tipo de datos tienen el valor en memoria al que podemos acceder por un identificador.
  • El manejo de las expresiones regulares en JS se manejan hacien las combinaciones con los caracteres en cadena, en Js estas expresiones regulares también son objetos. Pueden componerse por un carácter simple o también pueden ser caracteres combinados ya sean simples y especiales. Ejemplo: exec(), test(), que es de regExp

¿Cómo son todas las formas de declaración de funciones en JS?.

Hay 3 formas de declarar una función en JS, son: Por declaración, por expresión y por conductor de objetos.

  1. Por declaración: Este tipo de declaración de funciones es el más utilizado y fácil. Esté consiste en declarar la función con un nombre y declarar su parámetro de entrada dentro de un paréntesis.
  2. Por expresión: Este tipo de declaración de funciones consiste en guardar la función en una variable y luego ejecutar esa variable en forma de función.
  3. Por conductor de objetos: Este tipo de declaración de funciones no es muy recomendable ya que solo nos recuerda que las funciones también son un objeto en javascript.
